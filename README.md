## Projects

### 1. [Sales Insight Engine](./projects/sales-insight-engine/)

- Cleaned and analyzed 479 sales records across 24 months
- Multi-dimensional analysis: categories, regions, temporal trends
- Professional 4-panel visualization dashboard
- Complete business analysis report with actionable recommendations
- **Tech:** Python, Pandas, PostgreSQL, Matplotlib, Seaborn

**Key Features:**
- ETL pipeline with data validation
- Revenue analysis across product categories and regions
- Month-over-month growth tracking
- Business intelligence reporting

##  Getting Started

### Prerequisites

All projects use **[haashi_pkg](https://github.com/Haashiraaa/my-packages)**, a custom data engineering toolkit.

### Installation

```bash
# Clone the repository
git clone https://github.com/Haashiraaa/data-analysis-projects.git
cd data-analysis-projects

# Install haashi_pkg
pip install git+https://github.com/Haashiraaa/my-packages.git

# Navigate to specific project
cd projects/sales-insight-engine

# Install project dependencies
pip install -r requirements.txt

# Run the project
python main.py
```
