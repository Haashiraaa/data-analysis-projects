# Data_Viz

This directory contains data visualization projects using popular Python libraries such as Matplotlib and Plotly. Each project demonstrates techniques for exploring, analyzing, and visualizing datasets. The code showcases various chart types, customization options, and insights derived from data.

## Contents

- Python scripts for data visualization
- Sample datasets for practice and demonstration
- Visualizations including line charts, bar charts, scatter plots, and more

## Getting Started

1. Clone the repository:
   ```bash
   git clone https://github.com/Haashiraaa/data-viz-projects.git
   ```
2. Navigate to the `Data_Viz` directory and open scripts.

## Usage

Open and run the scripts to explore different visualization techniques. Modify the code to use your own datasets or experiment with customization.
